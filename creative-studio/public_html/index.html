MultiSpectral land Cover Analysis
Methodology
Data Acquisition and Preprocessing

The dataset from ESRI, comprising GeoTIFF tiles with nine LULC classes over six years, was rasterized and partitioned into manageable sets for efficient model training on limited hardware.

Model Creation and tuning

The model uses a three-layer LSTM architecture with dropout regularization and a final Softmax layer, optimized with the Adam optimizer and categorical cross entropy loss and utilizes early stopping for efficient convergence while training.

UI/UX Design and Deployment

For UI Development and Design, we used simpler Methods such as HTML, CSS and JavaScript, moreover we tried to integrate our model with more advanced and capable frameworks such as Reactjs and Angular.

Awesome Work

The report proposes a three-layer Long Short-Term Memory (LSTM) model to predict land use and land cover (LULC) changes using ESA Sentinel-2 imagery, achieving an average classification accuracy of 96.08%.

This model's high accuracy underscores its potential for aiding in environmental monitoring and policy planning by accurately forecasting LULC variations.

The research concludes with a call for further studies to enhance the model's performance and applicability in real-world scenarios.

Project Timeline

Literature Review and Data Acquisition

Conducted a thorough literature review and acquired ESA Sentinel-2 imagery for LULC analysis.

Data Preprocessing and Rasterization

Implemented rasterization techniques and one-hot encoding for data manipulation.

Model Development

Developed a three-layer Long Short-Term Memory (LSTM) model for predicting land cover changes.

Model Training and Validation

Trained the LSTM model on preprocessed data and validated its accuracy, achieving 96.08% classification accuracy.

Performance Evaluation

Evaluated model performance with various metrics and refined the model for improved reliability.

Web Application Development

Designed and built a user-friendly web application to access the model and analyze results.

Integration and Testing

Integrated the LSTM model into the web application and conducted comprehensive testing for functionality and usability.

Deployment and Documentation

Deployed the web application and prepared detailed documentation for end-users and future research opportunities.
